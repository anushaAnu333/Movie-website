## Features

- Browse movies in multiple sections (e.g. Popular, Top Rated, Upcoming)
- Search for movies by title
- Responsive grid layout for desktop & mobile
- Movie cards with poster, title, rating, and overview
- Easy-to-extend component architecture

---

## Tech Stack

- **Framework:** React  
- **Bundler:** Vite  
- **UI:** CSS (or your preferred styling) + Material-UI Icons  
- **Data:** Fetches movie data from an external API (e.g. TMDB)  
- **Deployment:** Vercel
